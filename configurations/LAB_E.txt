1. Configurations de base 
NB : j’ai utilisé un mot de passe très faible ici pour le line console et vty mais il est déconseiller de le faire donc dans un vrai réseau il ne faut jamais utiliser ce genre de mot de passe et jamais le divulguer. 

1.1. Passer en mode privilégié  
Router>enable 

1.2. Passer en mode de configuration globale
Router#configure terminal 

1.3. Renommer le routeur
Router(config)#hostname LAB_E

1.4. Attribution de mot passe pour l’accès par console et ssh ou telnet et pour passe en mode privilégie 
LAB_E(config)#line console 0
LAB_E(config-line)#password 1234
LAB_E(config-line)#exit
LAB_E(config)#line vty 0 4
LAB_E(config-line)#password 1234
LAB_E(config-line)#exit
LAB_E(config)#service password-encryption 
LAB_E(config)#enable secret 1234
LAB_E(config)#

1.5. Configuration des interfaces et activation du routeur

LAB_E(config)#interface gigabitEthernet 0/0
LAB_E(config-if)#ip address 192.168.6.254 255.255.255.0.
LAB_E(config-if)#no shutdown
LAB_E(config-if)#exit
LAB_E(config)#

LAB_E(config)#interface gigabitEthernet 0/1
LAB_E(config-if)#ip address 192.168.5.253 255.255.255.0
LAB_E(config-if)#no shutdown
LAB_E(config-if)#exit
LAB_E(config)#

.
LAB_E(config)#interface serial 0/1/0
LAB_E(config-if)#ip address 192.168.10.253 255.255.255.0
LAB_E(config-if)#no shutdown
LAB_E(config-if)#exit
LAB_E(config)#


2. Mise en œuvre du routage OSPF

LAB_E(config)#router ospf 1
LAB_E(config-router)#network 192.168.10.0 0.0.0.255 area 0
LAB_E(config-router)#network 192.168.5.0 0.0.0.255 area 0
LAB_E(config-router)#network 192.168.6.0 0.0.0.255 area 1
LAB_E(config-router)#exit
LAB_E(config)#
2.1. Création d’une interface de bouclage appelée loopback 0
LAB_E(config)#interface loopback 0
LAB_E(config-if)#ip address 5.5.5.5 255.255.255.255
LAB_E(config-if)#exit
LAB_E(config)#router ospf 1
LAB_E(config-router)#router-id 5.5.5.5
LAB_E(config-router)#end
LAB_E#clear ip ospf process
Reset ALL OSPF processes? [no]: yes
LAB_E#

3. Communication entre Aires
NB : pour que le virtual-link marche il faut supprimer le stub
LAB_E(config)#router ospf 1
LAB_E(config-router)#area 1 stub
LAB_E(config-router)#no area 1 stub
LAB_E(config-router)#area 1 virtual-link 6.6.6.6
LAB_E(config-router)#exit

4. Sauvegarder
