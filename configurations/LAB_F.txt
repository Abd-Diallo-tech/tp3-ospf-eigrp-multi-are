1. Configurations de base 
NB : j’ai utilisé un mot de passe très faible ici (1234) pour les lines console et vty mais il est déconseiller de le faire donc dans un vrai réseau il ne faut jamais utiliser ce genre de mot de passe et jamais le divulguer. 

1.1. Passer en mode privilégié  
Router>enable 

1.2. Passer en mode de configuration globale
Router#configure terminal 

1.3. Renommer le routeur
Router(config)#hostname LAB_F

1.4. Attribution de mot passe pour l’accès par console et ssh ou telnet et pour passe en mode privilégie 
LAB_F(config)#line console 0
LAB_F(config-line)#password 1234
LAB_F(config-line)#exit
LAB_F(config)#line vty 0 4
LAB_F(config-line)#password 1234
LAB_F(config-line)#exit
LAB_F(config)#service password-encryption 
LAB_F(config)#enable secret 1234
LAB_F(config)#
1.5. Configuration des interfaces du routeur
LAB_F(config)#interface gigabitEthernet 0/0
LAB_F(config-if)#ip address 192.168.7.254 255.255.255.0
LAB_F(config-if)#no shutdown 
LAB_F(config-if)#exit
LAB_F(config)#
LAB_F(config)#interface gigabitEthernet 0/1
LAB_F(config-if)#ip address 192.168.6.253 255.255.255.0
LAB_F(config-if)#no shutdown
LAB_F(config-if)#exit
LAB_F(config)#
LAB_F(config)#interface serial 0/1/0
LAB_F(config-if)#ip address 192.168.8.253 255.255.255.0
LAB_F(config-if)#no shutdown
LAB_F(config-if)#exit
LAB_F(config)#



2. Mise en œuvre du routage OSPF
LAB_F(config)#router ospf 1
LAB_F(config-router)#network 192.168.7.0 0.0.0.255 area 2
LAB_F(config-router)#network 192.168.8.0 0.0.0.255 area 2 
LAB_F(config-router)#network 192.168.6.0 0.0.0.255 area 1 
LAB_F(config-router)#exit
LAB_F(config)#
2.1. Création d’une interface de bouclage appelée loopback 0

LAB_F(config)#interface loopback 0
LAB_F(config-if)#ip address 6.6.6.6 255.255.255.255
LAB_F(config-if)#exit
LAB_F(config)#router ospf 1
LAB_F(config-router)#router-id 6.6.6.6
LAB_F(config-router)#end
LAB_F#clear ip ospf process 
Reset ALL OSPF processes? [no]: yes
LAB_F#


3. Communication entre Aires

LAB_F(config)#router ospf 1
LAB_F(config-router)#area 1 virtual-link 5.5.5.5
LAB_F(config-router)#exit
LAB_F(config)#
