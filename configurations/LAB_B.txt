1. Configurations de base 
NB : j’ai utilisé un mot de passe très faible ici pour le line console et vty mais il est déconseiller de le faire donc dans un vrai réseau il ne faut jamais utiliser ce genre de mot de passe et jamais le divulguer. 

1.1. Passer en mode privilégié  
Router>enable 

1.2. Passer en mode de configuration globale
Router#configure terminal 

1.3. Renommer le routeur
Router(config)#hostname LAB_B

1.4. Attribution de mot passe pour l’accès par console et ssh ou telnet et pour passe en mode privilégie 
LAB_B(config)#line console 0
LAB_B(config-line)#password 1234
LAB_B(config-line)#exit
LAB_B(config)#line vty 0 4
LAB_B(config-line)#password 1234
LAB_B(config-line)#exit
LAB_B(config)#service password-encryption 
LAB_B(config)#enable secret 1234
LAB_B(config)#

1.5. Configuration des interfaces du routeur

LAB_B(config)#interface gigabitEthernet 0/0
LAB_B(config-if)#ip address 192.168.2.253 255.255.255.0
LAB_B(config-if)#no shutdown 
LAB_B(config-if)#exit
LAB_B(config)#
LAB_B(config)#interface gigabitEthernet 0/1
LAB_B(config-if)#ip address 192.168.3.254 255.255.255.0
LAB_B(config-if)#no shutdown
LAB_B(config-if)#exit
LAB_B(config)#
LAB_B(config)#interface serial 0/1/0
LAB_B(config-if)#ip address 192.168.10.254 255.255.255.0
LAB_B(config-if)#no shutdown
LAB_B(config-if)#exit
LAB_B(config)#


2. Mise en œuvre du routage OSPF
LAB_B(config)#router ospf 1
LAB_B(config-router)#network 192.168.3.0 0.0.0.255 area 0
LAB_B(config-router)#network 192.168.10.0 0.0.0.255 area 0
LAB_B(config-router)#exit
LAB_B(config)#
2.1. Création d’une interface de bouclage appelée loopback 0
LAB_B(config)#interface loopback 0
LAB_B(config-if)#ip address 2.2.2.2 255.255.255.255
LAB_B(config-if)#exit
LAB_B(config)#router ospf 1
LAB_B(config-router)#router
LAB_B(config-router)#router-id 2.2.2.2
LAB_B(config-router)#end
LAB_B#clear ip ospf process
Reset ALL OSPF processes? [no]: yes
LAB_B#

3. Activation du routage EIGRP entre les Aires OSPF
LAB_B(config)#router eigrp 100
LAB_B(config-router)#network 192.168.2.0 0.0.0.255
LAB_B(config-router)#no auto-summary 
LAB_B(config-router)#exit
LAB_B(config)#
LAB_B(config)#router eigrp 100
LAB_B(config-router)#redistribute ospf 1 metric 10000 100 255 1 1500
LAB_B(config-router)#exit
LAB_B(config)#router ospf 1
LAB_B(config-router)#redistribute eigrp 100 subnets
LAB_B(config-router)#exit
LAB_B(config)#end

4. Sauvegarder
